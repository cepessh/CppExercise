# BinaryOperations Task

Fill in the functions in `src/bits.cpp`:

- `set_bit`, `clear_bit`, `toggle_bit`, `get_bit`
- `rotate_left`, `rotate_right`
- `count_set_bits`, `to_binary_string`

As you implement each helper, uncomment the corresponding unit test in
`tests/bits_tests.cpp` to verify your solution.

## Build and Test

Configure and build just this exercise with:

```bash
cmake -S . -B build
cmake --build build
```

Then run `./build/bits_tests`.
